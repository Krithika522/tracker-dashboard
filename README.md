Tracker Dashboard

A simple and interactive React dashboard using Higher-Order Components (HOC) to track:

✅ Mouse Position

✅ Window Size

✅ Online/Offline Status

Styled with Tailwind CSS.

🚀 Live Demo
👉 Live Site https://tracker-dashboard-coral.vercel.app/
